GIT/GitHub Quick Reference

  • Git basics
  • $/appname> git init :initialize repo
    $/appname> git status [-s] :for abbreviated status
    $/appname> git add FILENAME1 [FILENAME2] [. (add all files)] :start tracking files
    $/appname> git commit -m 'comments for commit' :start commiting to main/side branch
    $/appname> git commit -a ' 'comments for commit' :start commit and add together
    $/appname> git remote add [origin] https://github.com/username/somename.git :create remote repository named "origin"
    $/appname> git push [origin] master :upload files from "origin rep" to master branch
    $/appname> git checkout [fileNAme] [. (all files)] :if you need to retrieve the previous check in version
